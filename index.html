Launchy-alias-addon by 2called-chaos

Place a link to the executeable and use your aliases for directories, commands and urls...

Tested on: - Windows XP SP3

Dependencies

Launchy (www.launchy.net)

Install

Extract the Launchy_alias_addon.exe to any location you want. Make a link to this file in directory which is monitored by Launchy. Then press you Launchy hotkey and enter the link name, press tab and then type |help or edit you aliases.conf with |aliases. e.g.: [ALT]+[SPACE] => link [TAB] => |help => [ENTER]
